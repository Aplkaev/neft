Постановка задачи

Постановка задачи: Разработать прототип приложения по приемке геолого-геофизических данных для загрузки в БД РН-КИН.

Данные должны грузиться в таблицу БД. В качестве тестовой допускается использование произвольной БД с возможностью переключения на БД Oracle.

В форму Web интерфейса файлы должны помещаться путем перетаскивания мышкой.

Содержимое файлов должно помещаться в бинарное поле таблицы.
Если файл представляет собой архив, то он должен на лету разархивироваться и каждый файл из архива должен записываться отдельной строкой в таблицу.

Кроме того, должна заполняться дополнительная информация из поля «Дополнительная информация» WEB формы.

Приложение должно быть реализовано на Python с WEB интерфейсом

Допускается использование уже реализованных компонент.

Входные данные: Файлы, почтовые сообщения. Визуализация решения: Интерфейс приложения для загрузки файлов, форма с логом загрузки.
